Film Flock API Documentation

Film Flock will be an application where users can save their favorite movies via genra/director, users will be able to sort favorite movies after creating an account.

Endpoints

Get All Movies

Endpoint: /movies

Description: Retrieve a list of all movies in the database.

Method: GET

Usage in Postman: Request: GET http://localhost:5500/movies Response: Body: [Array of Movie Objects]

Create a New User

Endpoint: /users

Description: Create a new user with the provided username, password, email, and birthday.

Method: POST

Usage in Postman: Request: POST http://localhost:5500/users Body: { "Username": "exampleUser", "Password": "examplePassword", "Email": "user@example.com", "Birthday": "2000-01-01" } Response: Status: 201 Created Body: { "Username": "exampleUser", "Email": "user@example.com", "Birthday": "2000-01-01" // Additional user details }
