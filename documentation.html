Movie API Documentation (Film Flock)

Film Flock will be an application where users can save their favorite movies via genra/director, users will be able to stock a list of their favorite movies after creating an account.

Authentication note

Authentication is required for certain endpoints. Use the provided token in the Authorization header, and place user token in Bearer token section for authhencation purpouses.

Endpoints

1. Get All Movies

Retrieve a list of all movies.

GET https://nameless-basin-66959-08ab77b73096.herokuapp.com/movies

2. Get Movies by Genre

Retrieve movies based on a specific genre.

GET https://nameless-basin-66959-08ab77b73096.herokuapp.com/movies/details/genre/:genre

Replace :genre with the desired genre, ex., action, drama, comedy.

3. Get Movies by Director

Retrieve movies based on a specific director.

GET https://nameless-basin-66959-08ab77b73096.herokuapp.com/movies/details/director/:director

Replace :director with the name of the director.

4. Update User Info

Allow users to update their user information (username).

PATCH https://nameless-basin-66959-08ab77b73096.herokuapp.com/users/:userId

5. Add Movie to Favorites

Add a movie to a user's list of favorite movies.

POST https://nameless-basin-66959-08ab77b73096.herokuapp.com/users/:userId/favorites/:movieId

Replace :userId with the user's ID and :movieId with the movie's ID.

6. Remove Movie from Favorites

Remove a movie from a user's list of favorite movies.

DELETE https://nameless-basin-66959-08ab77b73096.herokuapp.com/users/:userId/favorites/:movieId

Replace :userId with the user's ID and :movieId with the movie's ID.

7. Create User

Create a new user account.

POST https://nameless-basin-66959-08ab77b73096.herokuapp.com/users

use body in json format to create a new user below is an exaple outline for creating users

Body: { "Username": "exampleUser", "Password": "examplePassword", "Email": "user@example.com", "Birthday": "2000-01-01" }

User name must be 5 or more characters

8. Get All Users

Retrieve a list of all users. Requires authentication, ALL usernames must be 5 chharacters or more.

GET https://nameless-basin-66959-08ab77b73096.herokuapp.com/users

9. Get User by Username

Retrieve user information based on the username. Requires authentication.

GET https://nameless-basin-66959-08ab77b73096.herokuapp.com/users/:Username

Replace :Username with the username.

10. Update User Information

Update user information based on the username. Requires authentication.

PUT https://nameless-basin-66959-08ab77b73096.herokuapp.com/users/:Username

Replace :Username with the username.

11. Delete User

Delete a user account based on the username. Requires authentication.

DELETE https://nameless-basin-66959-08ab77b73096.herokuapp.com/users/:Username

Replace :Username with the username.

Notes

  • Ensure you include the required authentication token in the bearer token section on Postman 2 for secured endpoints.
  • For endpoints requiring parameters, replace placeholders like :userId with accurate and real values.
  • Errors and status codes will be returned in case of issues. Check the response body for details.
